# brendan eich

![Portrait](images/portrait.jpg)
*Brendan Eich — created JavaScript in 10 days. Those 10 days now power virtually every website on Earth. Some say the technical debt from that sprint is still being paid off.*

**Field:** Programming Languages, Web Technologies

- **Lifespan:** b. 1961
- **Key contribution:** JavaScript programming language, Mozilla/Firefox, Brave browser
- **Impact:** JavaScript is the most widely deployed programming language in history — it runs in every web browser on every device

## Biography

### Early Life & Education
Born in Pittsburgh, Pennsylvania. Studied mathematics and computer science at Santa Clara University (BS), then got a master's degree from the University of Illinois at Urbana-Champaign. Worked on networking and kernel programming at Silicon Graphics and MicroUnity.

### Career
Joined Netscape Communications in April 1995 with the promise of putting "Scheme in the browser." Management had other ideas — they wanted something that looked like Java (for marketing synergy). In **10 days** in May 1995, Eich created "Mocha," which became "LiveScript," which became "JavaScript." It has nothing to do with Java except the name, which was purely a marketing decision.

Despite the rushed timeline, Eich managed to smuggle in some genuinely interesting ideas: first-class functions, closures, prototype-based inheritance (from Self), and dynamic typing. Also some questionable ones: automatic semicolon insertion, `==` type coercion, `typeof null === 'object'`.

Co-founded Mozilla in 1998, helped create Firefox. Briefly served as Mozilla CEO in 2014 (resigned after 11 days due to controversy over a political donation). Founded Brave Software in 2015, creating the Brave browser with built-in ad blocking and the Basic Attention Token (BAT) cryptocurrency.

### Later Life
CEO of Brave Software. Still involved in web standards and browser technology. Has a complicated relationship with JavaScript — he's both its creator and one of its most candid critics.

## Major Contributions

### 1. JavaScript (1995)
- **Year:** 1995 (created in 10 days in May)
- **Context:** Netscape needed a scripting language for the browser; had to "look like Java"
- **Technical Details:** Dynamic, weakly typed, prototype-based. First-class functions, closures (from Scheme). Prototype inheritance (from Self). Event-driven. Single-threaded with an event loop. Over the years: ES6/ES2015 added classes, arrow functions, promises, modules, template literals. Node.js (2009) brought it to servers.
- **Impact:** THE language of the web. Runs in every browser. Node.js made it a server language too. React, Angular, Vue, Next.js — the entire frontend ecosystem. Electron apps (VS Code, Discord, Slack). NPM is the largest package registry in the world. More developers know JavaScript than any other language.

### 2. Mozilla / Firefox
- **Year:** 1998-2014
- **Context:** After the "browser wars," Netscape open-sourced its code
- **Technical Details:** Co-founded the Mozilla project. Helped create Firefox, which broke Internet Explorer's monopoly and pushed for web standards.
- **Impact:** Firefox was crucial for web standards adoption. Without it, we might still be stuck with IE6.

### 3. Brave Browser & BAT (2015)
- **Year:** 2015
- **Context:** Privacy concerns and the broken ad model of the web
- **Technical Details:** Chromium-based browser with built-in ad/tracker blocking. Basic Attention Token (BAT) for rewarding content creators. Privacy-focused.
- **Impact:** Pioneered the privacy-first browser model. Over 60 million monthly active users.

## Influence & Legacy

### Direct Influence
JavaScript influenced TypeScript, CoffeeScript, Dart. Node.js created a full-stack JavaScript ecosystem. JSON (JavaScript Object Notation) became the universal data interchange format.

### Indirect Influence
Proved that a "good enough" language with ubiquitous deployment beats a "perfect" language with limited reach. The browser became a universal application platform because of JavaScript.

### Modern Relevance
JavaScript/TypeScript is the most popular language on GitHub. NPM has over 2 million packages. The entire web frontend and much of the backend runs on JS. WebAssembly (which Eich also championed) is expanding what browsers can do. Every interactive website you've ever used runs JavaScript.
