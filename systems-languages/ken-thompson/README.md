# ken thompson

![Portrait](images/portrait.jpg)
*Ken Thompson — co-created Unix, wrote the first computer chess program, and later co-invented Go. The kind of person who builds epoch-defining systems as side projects.*

**Field:** Operating Systems, Programming Languages

- **Lifespan:** b. 1943
- **Key contribution:** Unix, B language (precursor to C), regular expressions in computing, UTF-8 (with Rob Pike), Go language
- **Impact:** Unix and its descendants run the modern internet, smartphones, and virtually all servers

## Biography

### Early Life & Education
Born in New Orleans, Louisiana. Studied electrical engineering and computer science at UC Berkeley (BS and MS). Got hooked on programming early.

### Career
Joined Bell Labs in 1966. Worked on the Multics project — when Bell Labs pulled out, Thompson and Dennis Ritchie decided to build their own OS. Unix started because Thompson wanted to play Space Travel (a game he'd written for Multics) on a PDP-7. Designed the B programming language, which Ritchie later evolved into C. Wrote the first Unix text editor (ed), the first grep implementation. Created the first computer chess program — "Belle" — which was a world champion. Also implemented regular expressions in computing software.

In the 1990s, co-designed the Plan 9 operating system at Bell Labs. Co-created UTF-8 encoding with Rob Pike over a dinner napkin. After retiring from Bell Labs, joined Google where he co-created the Go programming language with Rob Pike and Robert Griesemer.

Famous for the "Reflections on Trusting Trust" Turing Award lecture (1984), where he demonstrated you can't fully trust any software, including compilers — a mind-bending revelation about computer security that remains relevant today.

### Later Life
Currently at Google. Still programming. Still building things. The man doesn't stop.

## Major Contributions

### 1. Unix Operating System (1969, with Dennis Ritchie)
- **Year:** 1969
- **Context:** After Multics fell apart, Thompson built Unix on a PDP-7 in about three weeks
- **Technical Details:** Pioneered "everything is a file" philosophy, hierarchical file system, shell, pipes. Simple, composable tools.
- **Impact:** Direct ancestor of Linux, macOS, iOS, Android, BSD. The most influential OS ever created.

### 2. B Programming Language (1969)
- **Year:** 1969
- **Context:** Needed a higher-level language for Unix development
- **Technical Details:** Derived from BCPL, simplified for the PDP-7. Ritchie later extended it into C.
- **Impact:** Bridge between BCPL and C; enabled the creation of C.

### 3. Regular Expressions in Software (1968)
- **Year:** 1968
- **Context:** Implemented regex matching in the QED text editor
- **Technical Details:** Brought Ken's implementation of regular expressions into practical computing tools. Led to grep ("global regular expression print").
- **Impact:** Every programmer uses regex. It's in every text editor, every programming language, every search tool.

### 4. Computer Chess — Belle (1970s–1980s)
- **Year:** 1970s-1980s
- **Context:** Built a chess-playing computer with Joe Condon
- **Technical Details:** Custom hardware for chess. World Computer Chess Champion. First machine to achieve master-level rating.
- **Impact:** Pioneered hardware-accelerated game AI.

### 5. UTF-8 (1992, with Rob Pike)
- **Year:** 1992
- **Context:** Needed a backward-compatible Unicode encoding
- **Technical Details:** Designed on a placemat at a New Jersey diner. Variable-width encoding compatible with ASCII.
- **Impact:** UTF-8 is THE encoding of the web. Over 98% of web pages use it.

### 6. Go Programming Language (2007–2009, with Rob Pike and Robert Griesemer)
- **Year:** 2009
- **Context:** Frustrated by C++ compile times at Google
- **Technical Details:** Simple, compiled, concurrent, garbage-collected. Goroutines and channels (influenced by Hoare's CSP). Fast compilation.
- **Impact:** Widely used for cloud infrastructure, microservices, DevOps tools (Docker, Kubernetes are written in Go).

## Influence & Legacy

### Direct Influence
Unix, B, regex, UTF-8, Go — each one of these alone would be a career-defining achievement. Thompson has about six of them.

### Indirect Influence
The Unix philosophy shaped how generations of programmers think. Regex is universal. UTF-8 made the multilingual web possible. Go is reshaping cloud infrastructure.

### Modern Relevance
Go is one of the fastest-growing languages. UTF-8 dominates the web. Linux (Unix's descendant) runs most of the internet. Thompson's influence is everywhere, all the time.
